# Validation 1 – Smart Hospital Navigation

## Related Feature
Feature 1 – Smart Hospital Navigation

## Validation Goal
To check whether users can find hospital departments easily using the navigation feature.

## Test Scenario
A user opens the Sukoon application and searches for a hospital department, such as Radiology, Emergency, or Reception. The user follows the provided directions.

## Validation Method
- Usability test with sample users
- Observation of user actions
- Short feedback questions after the test

## Success Criteria
- The user can find the department search function without help.
- The user can select a department successfully.
- The user understands the displayed directions.
- The user can complete the task in less than two minutes.
- At least 80% of test users rate the navigation as clear and helpful.

## Questions for Users
1. Was it easy to find the department search?
2. Were the directions clear?
3. Did the design feel calm and simple?
4. What part was confusing?
5. What would you improve?

## Expected Result
The navigation feature should reduce confusion and help users move through the hospital more confidently.
